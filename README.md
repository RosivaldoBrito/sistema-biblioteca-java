# Projeto - Sistema de Biblioteca

 Objetivo

Desenvolver um sistema de biblioteca totalmente em Java utilizando o console.

O aplicativo deve permitir a gestão de livros, usuários e empréstimos.

---

 Funcionalidades

# Livros

* registrar livro
* modificar livro
* excluir livro
* listar livros
* procurar por título
* procurar por autor
* procurar por categoria
* verificar disponibilidade

---

# Usuários

* cadastrar usuário
* modificar usuário
* remover usuário
* listar usuários

---

# Empréstimos

* realizar empréstimo de livro
* devolver livro
* bloquear empréstimo caso o livro esteja emprestado
* impedir empréstimo para usuário que não existe
* exibir data do empréstimo
* mostrar previsão de devolução
* calcular dias de atraso

---

# Persistência

Quando o aplicativo for finalizado:

* armazenar todas as informações em arquivos

Quando for iniciado novamente:

* carregar todas as informações novamente.

---

# Menu

Um menu extenso em loop.

Exemplo:

```
1 - Livros
2 - Usuários
3 - Empréstimos
4 - Relatórios
0 - Sair
```

Cada uma das opções tem seus submenus.

---

# Relatórios

Exemplos:

* livros que estão emprestados
* livros disponíveis
* usuários com livros emprestados
* livros mais requisitados
* quantidade de livros por categoria
